# Vue 3 + TypeScript + Pug Starter

Шаблон для разработки на Vue 3 с использованием TypeScript, синтаксиса шаблонов Pug и линтинга ESLint.

Просто 
* склонируй
* смонтирую сюда src
* и работай

## Особенности сборки

* **Фреймворк:** Vue 3 (Composition API)
* **Языки:** TypeScript + Pug
* **Инструмент сборки:** Vite
* **Качество кода:** ESLint (Flat Config) + проверка типов через `vue-tsc`
